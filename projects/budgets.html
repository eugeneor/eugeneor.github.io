Tracking Matter Budgets

Clio | April to June 2015

Back Next Project

Problem

Customers want to add a budget when dealing with their matters. They want to be able to measure time and expenses against a budgeted amount. To maintain accountability, notifications were also required if a threshold is reached or if a budget is edited by someone in their firm.

Solution

Using the existing structure we have in place for the dashboard, we built a progress bar widget that would let customers visualize their time entries and expenses against a set budget. Each time the customer adds time to a matter, the bar would fill up more to reflect the addition. Email notifications were sent to selected individuals if certain triggers were hit.
